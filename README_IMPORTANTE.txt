🚀 COMO USAR?

PASSO 1: Extrair o ZIP
- Clique direito no arquivo
- Selecione "Extrair aqui" ou "Extract All"

PASSO 2: Copiar os Arquivos para o Seu Projeto
- Abra a pasta "portal-ti-alteracoes"
- Copie os 5 arquivos HTML e JS:
  * script.js
  * Inicio.html
  * Materiais.html
  * Contacto.html
  * sobre.html

PASSO 3: Colar no Seu Projeto
- Vá para a pasta do seu projeto local:
  C:\Users\[seu-usuario]\Desktop\Portal-do-Grupo-de-Tecnologias-de-Informa-o\
  
- Cole os arquivos (substituindo os antigos)

PASSO 4: Testar Localmente
- Abra o terminal na pasta do projeto
- Execute: python3 -m http.server 8000
- Abra http://localhost:8000 no navegador

PASSO 5: Fazer Commit e Push
- No terminal, execute:
  git add .
  git commit -m "feat: implementar interações JavaScript, corrigir links e ajustes finais"
  git push origin feature/frontend-fixes-links-js

PASSO 6: Abrir Pull Request
- Vá para GitHub
- Clique em "Pull requests"
- Clique em "New pull request"
- Selecione feature/frontend-fixes-links-js → dev
- Descreva as alterações
- Clique em "Create pull request"
